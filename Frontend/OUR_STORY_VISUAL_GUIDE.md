## Responsive Behavior

### Breakpoint Changes

**Mobile (<640px)**
- Hero: Smaller text (4xl)
- Story: Single column, image on top
- Values: Single column, stacked
- Gallery: 1 column
- Stats: Smaller numbers (3xl)
- Buttons: Full width, stacked

**Tablet (640px - 1023px)**
- Hero: Medium text (5xl)
- Story: Two columns
- Values: May stack or show 2 columns
- Gallery: 2 columns
- Stats: Medium numbers (4xl)
- Buttons: Side by side

**Desktop (≥1024px)**
- Hero: Large text (6xl)
- Story: Two columns, side by side
- Values: Three columns
- Gallery: 4 columns
- Stats: Large numbers (4xl)
- Buttons: Side by side with spacing

## Image Specifications

### Hero Image
- Dimensions: 1920x800px
- Format: JPEG/WebP
- Size: < 300KB
- Position: Center

### Featured Image
- Dimensions: 800x800px
- Format: JPEG/WebP
- Size: < 200KB
- Aspect: Square

### Gallery Images
- Dimensions: 600x600px
- Format: JPEG/WebP
- Size: < 150KB each
- Aspect: 1:1 (Square)

### Decorative Images
- Dimensions: 300x300px
- Format: JPEG/WebP
- Size: < 100KB
- Aspect: Square

## Interactive Elements

### Hover Effects
```
Gallery Images:
- Scale: 100% → 110%
- Overlay: 0% → 40% opacity
- Title: opacity 0 → 1
- Transition: 300ms

Buttons:
- Background color change
- Smooth transition
- Cursor: pointer

Value Cards:
- Subtle lift (optional)
- Shadow increase (optional)
```

### Click Actions
```
Gallery Image:
→ Opens lightbox modal
→ Shows full-size image
→ Displays title

Close Button:
→ Closes lightbox
→ Returns to gallery

Outside Click:
→ Closes lightbox
```

## Accessibility Features

### Keyboard Navigation
- Tab through gallery images
- Enter to open lightbox
- Escape to close lightbox
- Focus visible on all interactive elements

### Screen Reader
- Alt text on all images
- Descriptive headings
- Proper semantic structure
- ARIA labels where needed

### Visual
- High contrast text
- Readable font sizes
- Clear focus indicators
- Sufficient spacing
