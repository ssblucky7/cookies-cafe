At Cookies Café, we believe that cookies are more than just a

ingredients.

welcoming space where community connections flourish.

To become the most beloved cookie café in the region, known for 
our commitment to quality, innovation, and community impact,

## Customization Guide

### Update Team Members
Edit the `team` array in `AboutUs.jsx`:
```javascript
{
  name: 'New Member',
  role: 'Position',
  image: 'image-url',
  bio: 'Biography text...'
}
```

### Add Milestones
Edit the `milestones` array:
```javascript
{
  year: '2025',
  event: 'Event Name',
  description: 'Description...'
}
```

### Modify Values
Edit the `values` array:
```javascript
{
  icon: <FiIcon />,
  title: 'Value Name',
  description: 'Description...'
}
```

### Change Mission/Vision
Update the text directly in the JSX
